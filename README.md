
# 浏览器端电路图绘制器

一个运行在浏览器中、**完全离线可用**的电路图绘制与仿真工具。专为中学电子课和入门爱好者设计，支持拖拽元件、连接引脚、实时计算电路通断与分压，并通过灯泡亮度直观反馈电流大小。

## ✨ 功能特性

- **🎨 可视化画布**：基于 Konva 的高性能画布，支持拖拽、缩放、平移
- **🔧 元件库**：内置电池、电阻、开关、灯泡四种基础元件
- **🔗 可视化连线**：点击引脚即可拖拽连线，支持直角布线
- **⚡ 实时电路仿真**：
  - 自动检测电路是否形成完整回路
  - 基于图搜索的串并联电路求解
  - 实时计算各支路电流、节点电压
  - 灯泡按电流大小显示不同亮度
- **💾 本地持久化**：基于 IndexedDB 保存电路，下次打开继续编辑
- **📸 图片导出**：一键导出当前电路图为 PNG 图片
- **📚 示例电路**：内置 4 个经典示例电路，可直接加载学习

## 🏗️ 架构设计

项目严格遵循**三层分离**原则，元件库定义、画布交互、电路求解三者完全解耦：

```
┌─────────────────────────────────────────────────────┐
│                    UI 层 (React)                      │
│  Toolbar | ComponentLibrary | PropertyPanel          │
└─────────────┬───────────────────────┬───────────────┘
              │                       │
              ▼                       ▼
┌─────────────────────────┐  ┌────────────────────────┐
│    画布层 (Konva)        │  │  状态管理层 (Zustand)   │
│  拖拽 / 连线 / 缩放       │  │  元件 / 导线 / 结果     │
└─────────────────────────┘  └──────┬─────────────────┘
                                   │
            ┌──────────────────────┼──────────────────────┐
            ▼                      ▼                      ▼
   ┌────────────────┐   ┌──────────────────┐   ┌────────────────┐
   │  元件定义层     │   │  电路求解器       │   │  存储层          │
   │ components-def │   │  solver (独立)    │   │  db (IndexedDB) │
   └────────────────┘   └──────────────────┘   └────────────────┘
```

### 模块独立性说明

| 模块 | 路径 | 依赖 | 可单独测试 |
|------|------|------|------------|
| 元件定义 | `src/components-def/` | 仅类型 | ✅ |
| 电路求解器 | `src/solver/` | 类型 + 元件定义 | ✅ 独立 Vitest 测试 |
| 画布层 | `src/components/Canvas/` | 元件定义 + Store | 通过 React 测试 |
| 状态管理 | `src/store/` | 求解器 + 元件定义 | ✅ |
| 存储层 | `src/db/` | 类型 | ✅ |

## 📖 使用指南

### 基础操作

1. **添加元件**：从左侧元件库拖拽元件到画布
2. **移动元件**：在画布上直接拖动元件
3. **连接导线**：点击元件上的蓝色圆点（引脚），再点击另一元件的引脚完成连线
4. **取消连线**：右键点击或点击空白处取消
5. **切换开关**：点击开关元件即可切换闭合/断开
6. **调整参数**：选中元件后，在右侧属性面板拖动滑块调整电压/电阻
7. **缩放/平移**：鼠标滚轮缩放，画布空白处拖拽平移
8. **删除**：选中元件或导线后，点击右侧面板的删除按钮

### 加载示例电路

点击顶部工具栏的 **「示例」** 按钮，可加载以下经典电路：

| 示例 | 说明 |
|------|------|
| 最简单的灯泡电路 | 电池 → 灯泡，最基础的回路 |
| 开关控制灯泡 | 加入开关，演示通断控制 |
| 串联电阻分压 | 电阻与灯泡串联，灯泡变暗 |
| 并联双灯泡 | 两个灯泡并联，亮度互不影响 |

### 保存与打开

- **保存**：点击「保存」按钮，输入电路名称后存入浏览器本地 IndexedDB
- **打开**：点击「打开」查看并加载已保存的电路
- **导出图片**：点击「导出图片」将当前画布保存为 PNG

## 🧠 求解器算法说明

电路求解位于 `src/solver/index.ts`，完全独立于 UI，可通过 `solveCircuit()` 函数单独调用。

### 核心算法

1. **节点合并（Union-Find）**：将通过导线和闭合开关连接的引脚合并为同一节点
2. **通路检测（BFS）**：从电池正极出发，判断是否能经过被动元件（电阻/灯泡）到达负极
3. **路径枚举（DFS）**：找出所有从正极到负极的独立电流路径
4. **欧姆定律计算**：对每条路径按总电阻计算电流，经过同一元件的多条路径电流叠加

### 求解器接口

```typescript
import { solveCircuit } from '@/solver';

const result = solveCircuit({
  components: [...],   // 元件列表
  wires: [...],        // 导线列表
});

// 返回：
// {
//   isConnected: boolean,           // 电路是否导通
//   nodeVoltages: Record<string, number>,   // 各节点电压
//   branchCurrents: Record<string, number>, // 各元件电流
//   bulbBrightness: Record<string, number>, // 灯泡亮度 0~1
//   errors: string[],                // 错误信息
// }
```

## 🔌 扩展元件

新增元件只需修改两个地方（无需改动画布和求解器）：

1. 在 `src/components-def/index.ts` 中添加元件定义（引脚、参数、尺寸）
2. 在 `src/components/Canvas/CircuitNode.tsx` 的 `renderComponentBody()` 中添加渲染逻辑
3. 如元件有特殊电学特性，在 `src/solver/index.ts` 中扩展处理
